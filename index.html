Vectoria

A lightweight, client-side vector search engine for the browser.

Vectoria brings powerful vector search capabilities directly to the client side. Built with HNSW and IndexedDB, it allows you to build semantic search, RAG, and recommendation systems that run entirely in the user's browser.

Try the Demo Performance Test View on GitHub

Client-Side Search

Runs entirely in the browser using the HNSW algorithm. No server-side vector database required.

Local & Private

Run transformer models locally using @huggingface/transformers (ONNX/WASM) for complete privacy.

Persistent

Automatically saves your index and documents to IndexedDB, ensuring data persists between sessions.

Installation

npm install vectoria
